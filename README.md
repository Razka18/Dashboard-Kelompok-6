# Dashboard Sales Report (Kelompok 5)

Proyek ini adalah Dashboard interaktif untuk pelaporan penjualan yang dikembangkan menggunakan **Streamlit** (Python). 

## 💻 Panduan Menjalankan di Komputer Lokal

Jika ingin menjalankan atau memodifikasi *dashboard* ini di komputer Anda sendiri, silakan ikuti langkah-langkah mudah berikut:

### Prasyarat
Pastikan Anda sudah menginstal Python (minimal versi 3.8+).

### 1. Install Dependencies
Untuk menginstal semua pustaka Python yang dibutuhkan, buka terminal (CMD/PowerShell) di folder proyek ini dan jalankan:

```bash
py -m pip install -r requirements.txt
```
*(Catatan: Gunakan `py` di Windows. Jika Anda menggunakan Mac/Linux, gunakan `python3` atau `pip3`)*

### 2. Menjalankan Dashboard Streamlit (Versi Interaktif Python)
Untuk melihat dashboard versi asli Streamlit yang memproses data secara langsung, jalankan perintah ini:

```bash
py -m streamlit run layout.py
```
Dashboard akan otomatis terbuka di browser pada URL **http://localhost:8501**.
